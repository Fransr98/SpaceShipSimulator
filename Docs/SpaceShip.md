# Funcionamiento clase  SpaceShip 🚀

#### Creacion de una nave espacial mediante Java.

## **Objetivos a realizar:** 📋

===================================================

### **1º- Crear una clase tipo SpaceShip que contenga los siguientes elementos:**📌

#### -Nombre.
#### -Matrícula galáctica.
#### -Aceleración.
#### -Velocidad X.
#### -Velocidad Y.
#### -Coordenada X.
#### -Coordenada Y.
#### -Dirección Y.
#### -Dirección X.
